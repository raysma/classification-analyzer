# Changelog

## 2026-05-14

### Features

- **USPSA member lookup**: enter any USPSA member number to automatically fetch and display your full classifier history from uspsa.org.
- **Division tabs**: view classifier history and stats per division; tab labels show the classifier count for each division.
- **Classification summary**: shows your current class letter, classification percentage, gap to the next class threshold, and all-time highest percentage in the division.
- **Score chart**: classification percentage plotted over time with color-coded dots by class band (GM/M/A/B/C/D) and horizontal threshold reference lines.
- **Class-up insights**: calculates the minimum uniform average required across the next 1–5 classifiers to reach the next class.
- **What-if simulator**: add hypothetical future scores and see your projected classification percentage and class update in real time.
- **Classifier history table**: full sortable history with color-coded percent column, flag descriptions on hover, and highlighted Y / dimmed F rows matching the rolling-window calculation.
- **Manual paste**: paste your classifier history table directly from USPSA.org for any division when the automatic lookup is unavailable.
- **Dark mode**: three-way Light / Auto / Dark theme toggle persisted across sessions; Auto follows the OS preference.
